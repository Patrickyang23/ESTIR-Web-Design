solid electrolyte interface (layer)

A passivating type layer formed on lithium in contact both with liquid and polymer electrolytes when used as the negative electrode in a battery. It is generally considered that this layer makes the practical lithium batteries possible because lithium itself is too reactive to be stable in contact with the electrolytes. This layer acts as an interface between the metal and the solution and has the properties of a solid electrolyte. Abbreviated as "SEI".
